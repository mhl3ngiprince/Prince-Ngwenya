# Prince-Ngwenya

This repository is now scaffolded as a small React (Vite) portfolio website for Mhlengie Prince Ngwenya.

Quick start (Windows PowerShell):

```powershell
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

Files added:
- `package.json` — scripts and dependencies (Vite + React)
- `public/index.html` — app entry HTML
- `src/main.jsx`, `src/App.jsx` — React entry and app container
- `src/components/*` — header, hero, about, experience, projects, contact, footer
- `src/index.css` — minimal styling adapted from the existing design

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mhl3ngiprince/Prince-Ngwenya.git
   cd Prince-Ngwenya
   ```

2. **Install dependencies**  
   (If using Node.js and a package manager like npm or yarn)
   ```bash
   npm install
   ```
   _or_
   ```bash
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
   _or_
   ```bash
   yarn start
   ```

## Usage
